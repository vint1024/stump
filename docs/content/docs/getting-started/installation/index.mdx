---
title: Installation
description: How to install Stump and get it up and running
---

This section primarily refers to installation of the Stump server, which has three primary methods of installation:

- [Pre-built binaries](/docs/getting-started/installation/binaries)
- [Docker](/docs/getting-started/installation/docker)
- [Manual build from source](/docs/getting-started/installation/source)

Additionally, there are the following methods of installation that are not officially supported by the Stump developer, but may be maintained by community members:

- [Community packages](/docs/getting-started/installation/source#community-packages) (e.g., the AUR package)

If you're looking for installation instructions for other Stump clients (e.g., the mobile app), please refer to the [apps documentation](/docs/apps/mobile).
